Welcome to The Formatter's Lab!

In this lab, we will explore various HTML formatting tags and how they can be used to enhance the presentation of your web content.

Bold and Strong Tags

The <b> tag is used to make text bold, while the <strong> tag is used to indicate that the text is of strong importance. Both tags will visually make the text bold, but the <strong> tag also conveys semantic meaning to search engines and assistive technologies.

Example of bold text using the <b> tag: This text is bold.
Example of strong text using the <strong> tag: This text is strong and important.

Italic and Emphasis Tags

The <i> tag is used to italicize text, while the <em> tag is used to indicate emphasis. Similar to the bold and strong tags, both will visually italicize the text, but the <em> tag also conveys semantic meaning.

Example of italic text using the <i> tag: This text is italicized.
Example of emphasized text using the <em> tag: This text is emphasized.

Small and Mark Tags

The <small> tag is used to make text smaller, while the <mark> tag is used to highlight text.

Example of small text using the <small> tag: This text is small.
Example of marked text using the <mark> tag: This text is highlighted.

Deleted Text Tag

The <del> tag is used to indicate text that has been deleted from the document.

Example of deleted text using the <del> tag: This text has been deleted.

Subscript and Superscript Tags

The <sub> tag is used to create subscript text, while the <sup> tag is used to create superscript text.
